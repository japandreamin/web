# Japan Dreamin' Website

## Content Development
To run local server, run
```
hugo server
```
And then open `http://localhost:1313/`.

## Style
In `themes/ohana-theme-hugo`, to watch Sass files, run
```
npm run start
```

To build the files, run
```
npm run build
```
